# MapControl
These examples show all the map navigation controls on the map and how to customize different option settings.

## Style control


<MapControl controls={[controls[0]]}/>
<Source code={`
import {
  ControlOptions,
  AzureMapsProvider,
  AzureMap,
} from 'react-azure-maps';

<AzureMapsProvider>
  <div className="defaultMap">
    <AzureMap
      options={your_options}
      controls={[
        {
          controlName: 'StyleControl',
          controlOptions: { mapStyles: 'all' },
          options: { position: 'top-right' } as ControlOptions,
        },
      ]}
    />
  </div>
</AzureMapsProvider>;
`}/>

## Traffic control and Legend control

<MapControl controls={[controls[4], controls[5]]}/>

  ControlOptions,
  AzureMapsProvider,
  AzureMap,
